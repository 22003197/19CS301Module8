### AIM

To write python program to print the cube of each number on a separate line.

### ALORITHM

2. Read an integer n.

3. For each number i from 0 to n-1:
   a. Compute iÂ³ using i**3.
   b. Print the result.

4. End loop after reaching n-1.

```

for i in range(n):
    print(i**3)
```
